# Marks Exchange Brand Style

## Colors
- Primary dark: #18202B (navy) - main backgrounds
- Primary light: #FFFEEF (cream) - text, light backgrounds
- Accent positive: #22C55E (green) - positive values, CTAs
- Accent negative: #EF4444 (red) - negative values, alerts

## Visual Style
- Professional fintech aesthetic
- Minimal, clean compositions
- Currency symbols as decorative elements (USD, EUR, NGN, ARS, etc.)
- Global/international feel
- Elegant typography with italic serif for headlines

## Modes
- **Dark mode**: Navy (#18202B) background with cream (#FFFEEF) text
- **Light mode**: Cream (#FFFEEF) background with navy (#18202B) text

## Do
- Use the brand color palette consistently
- Include subtle currency symbol patterns or decorative elements
- Keep compositions clean and uncluttered
- Convey sophistication, trust, and professionalism
- Show global connectivity and international markets
- Use abstract representations of trading/finance

## Don't
- Use bright, flashy, or neon colors
- Overcrowd with too many elements
- Use casual, playful, or cartoonish imagery
- Include specific data/numbers (they will be inaccurate)
- Use generic stock photo aesthetics
- Include text that needs to be readable (use visual elements only)
